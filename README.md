# Takagi Works site

Static site for the Takagi Works projects using Jekyll.

## Running locally

```bash
gem install bundler jekyll   # once, if you do not have Jekyll
bundle exec jekyll serve     # run at http://localhost:4000
```

## Structure

- `index.md` – landing page with quick links to Takagi and Hako
- `takagi.md`, `hako.md` – project spotlight pages
- `blog/` – blog index powered by `_posts/`
- `_layouts/`, `assets/css/` – layouts and styling
